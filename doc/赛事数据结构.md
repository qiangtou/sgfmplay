##盘口信息

### 数据内容:
第三方赔率,买卖盘口数据 JSON字符串格式: 

    {
		c:0,
		d:[
			[交易项ID,参考赔率,买1赔率,买1数量,买2赔率,买2数量,买3赔率,买3数量,卖1赔率,卖1数量,卖2赔率,卖2数量,卖3赔率,卖3数量]
		],
		v:123,
		e:"",
		t:0(返回数据是全量或增量,1是全量，0是增量)
	}    
注:没有的话就空,删除是-1

###请求参数:
全局请求:当第一次进入页面时,或者到了全局请求的时间,发送该请求到后台,带上筛选条条件.
联赛请求:带上联赛ID+版本号+筛选条件.
赛事请求:带上赛事ID+版本号+筛选条件.

1. 需第一次数据加载正确并成功。
2. 通过交易项ID找对应的位置进行增量数据的修改。
